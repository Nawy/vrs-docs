# Quick start

Let's discover **Versolid in less than 5 minutes**.

## Commands

### Init
Initialize Versolid project in current folder: 
```bash
vrs init
```

### Status
Check status of Versolid project:
```bash
vrs status
```

### Commit
Create commit:
```bash
vrs commit new "<commit_name>"
```
